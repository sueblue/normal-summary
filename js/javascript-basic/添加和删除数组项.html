首先，要在数组末尾添加或删除一个项目，我们可以使用push()和pop()。
PS:push和pop即出栈入栈，先进后出，后进先出。
当方法调用完成时，将返回数组的新长度。 如果要将新数组长度存储在变量中，可以执行以下操作： var newLength = myArray.push('Bristol'); newLength;
从数组中删除最后一个项目就像在其上运行pop（）一样简单。 myArray.pop();
当方法调用完成时，将返回已删除的项目。 你也可以这样做： var removedItem = myArray.pop(); myArray; removedItem;
unshift()和shift()以完全相同的方式工作，只是它们在数组的开始处，而不是结尾。 首先unshift() — 尝试一下命令 myArray.unshift('Edinburgh'); myArray; 现在shift（）; 尝试这些！ var removedItem = myArray.shift(); myArray; removedItem;
